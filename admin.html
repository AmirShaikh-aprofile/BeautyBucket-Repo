BeautyBucket Admin

Manage products via Google Sheets

Admin Instructions (No coding)

  1. Create a Google Sheet using the provided template CSV files (products + dealers).
  2. Fill product rows (ProductName, Category, Details, MRP, SellingPrice1pc, SellingPrice5pc, Stock, ImageURL).
  3. From Google Sheets: File → Publish to web → choose the sheet and "Comma separated values (CSV)" → publish and copy the generated link.
  4. Open the frontend index.html (hosted on GitHub Pages) and replace the PRODUCTS_CSV_URL in the script with your CSV export link: 
    https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/export?format=csv&gid=0
  5. To edit products, open the Google Sheet directly. Changes will reflect on the site within a minute (Google caching).

Dealer Quotations (sheet)

Use the separate "dealers" sheet to add dealer-specific offers. The template includes formula columns to auto-calc Profit and Discount%.
